About This Site

This site was created by Yujin Johnson. It's a profile and fact website on the female KPOP group Lesserafim. Images from official instagram accounts and photoshoots are included to make it look more professional and official.

Bonus Marks

  • Some Javascript functionality

    The automatic gallery and the back-to-the-top button uses javascript for further funtionality and engagement of the website.

  • Use of an external CSS file to ensure a consistent look and feel of the site and easy changes to the appearance of the entire site.

    All the pages are linked to a style sheet. Because of some overlapping coding, the profile pages for the members have a separate stylesheet. This has been done to make styling easier and more regulated.

  • Overall appearance and consistency

    The overall appearance is very consistent with the header and navigation bar on all pages and similar formatting on the profile and song pages. There is only black and white (excluding the images) because those are the colours of the official logo and the website had been styled around that colour scheme. Everything follows the stylesheets which makes the font, colours, sixe, width, etc. regular. The formatting is regulated as well such as the centered header and navigation bar.
